# Laufey's Website | Desarrollador Front-End y diseñador

> Proyecto académico para la materia de Tópicos de Calidad para el Diseño de Software, enfocado en **buenas prácticas de Front-end** y diseño centrado en el usuario.

---

##  Desafío

El principal desafío fue la aplicación metodológica de los estándares de calidad de software dentro de un proyecto Front-end.

* **Problema Resuelto:** Creación de una maqueta funcional para la página oficial de la cantautora Laufey, priorizando el diseño responsivo y la maquetación semántica para asegurar el mantenimiento a futuro.
* **Objetivo de Calidad:** Implementar la estructura del proyecto basándose en principios de diseño limpio, nomenclatura consistente, y código legible (clean code).

---

## Solución y Características

* **Página Principal:** Implementación de un diseño inmersivo y una navegación optimizada para una experiencia de usuario fluida.
* **Sección Multimedia:** Integración de una galería interactiva con carrusel, permitiendo al usuario una vista ampliada del contenido visual.
* **Diseño Responsivo:** Compatibilidad total con dispositivos móviles y tablets.

* **Dominio Técnico:** Dominio de la maquetación semántica con HTML5 y organización avanzada de estilos con CSS para garantizar el mantenimiento a largo plazo.
